Unpublished Technical Work

    I wrote a simple model for the time-dependence of the Pfizer-BioNTech COVID-19 vaccine and analyzed it using a dynamic nested sampler in order to determine reentry date in the spring of 2021. I incorporated data from several published papers. Do not distribute: Pfizer-BioNTech COVID-19 vaccine model

    Extending my work on isotropic wavelet transforms, I have applied an anisotropic, two-dimensional, undecimated wavelet transform to gamma-ray data to search for signatures of dark matter annihilation with an enhanced signal to background ratio. This required surmounting some concrete technical challenges such as:
    • using a least-squares estimator, rather than the more common Poisson likelihood (which is a maximum-likelihood estimator), because entries were neither integer-valued nor even guaranteed to be positive definite
    • establishing a simulation-based covariance matrix with a sufficient number of iid simulations as to be invertible
    • regularizing the method of normal equations to incorporate data-driven priors on certain parameters
    This work is being prepared for publication, but a draft is available upon request.

Code

    dmdd is a python package that enables simple simulation and Bayesian posterior analysis of nuclear-recoil data from dark matter direct detection experiments for a wide variety of theories of dark matter-nucleon interactions. dmdd was developed in collaboration with Vera Gluscevic for use in arXiv:1506.04454, additionally with Moira Gresham, Annika H. G. Peter, and Kathryn M. Zurek.
